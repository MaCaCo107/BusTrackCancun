BusTrack Cancún

Tecnologías Utilizadas:
°Frontend: Android Studio (Kotlin), OpenStreetMap (osmdroid).
°Backend: Node.js, Express.
°Base de Datos: MongoDB Local.

Funcionalidades Actuales:
°Registro e Inicio de sesión de usuarios.
°Visualización de rutas trazadas en rojo (12f).
°Sistema de comentarios de la comunidad con globos de texto interactivos.
  
Cómo ejecutar el proyecto:
1.Iniciar MongoDB Local.

2.Ejecutar node server.js en la carpeta /backend-api.

3.Abrir /android-app en Android Studio y ejecutar en un emulador.
